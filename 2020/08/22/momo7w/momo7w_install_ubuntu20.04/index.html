准备 hub otg u盘键鼠

首先下载 http://cdimage.ubuntu.com/lubuntu/releases/20.04/release/lubuntu-20.04-desktop-amd64.iso

用新版的rufus烧录到sd卡 我一开始是烧到u盘的，结果安装过程中有问题，更换了u盘和hub问题依旧 最后在阿雪指教下换烧sd卡。
因为momo7w的efi是32位的，所以我们把bootia32.efi 拷贝到sd的 efi/boot里面。efi文件可以从群共享下载。
sd卡插入momo7w（下文m7），通电开机 按esc
选第一个 打开 live环境

4、配置自动连接wifi 路由器DHCP分批静态IP
因为默认是图形界面不登录不连接wifi的，我用不太到图形界面 所以这里设置一下自动wifi
网上教程多数是wpa_passphrase 或者开机脚本之类的其实没那么复杂

5、关闭Lxqt图形界面 三步操作缺一不可
修改sudo vi /etc/default/grub
注释掉 GRUB_CMDLINE_LINUX_DEFAULT
修改GRUB_CMDLINE_LINUX=”text”
添加或者去掉注释GRUB_TERMINAL=console

重启之前最好看一下电源的设置，部分Ubuntu有开启休眠等。我这里是24小时开机的所以不能让他休眠了
能节省150M左右内存
恢复方法
直接sudo systemctl set-default graphical.target 就可以了，如果看不惯开机的一串命令行 就按照下面修改

添加或者去掉注释GRUB_CMDLINE_LINUX_DEFAULT=”quiet splash”
GRUB_CMDLINE_LINUX=”text” 改为 GRUB_CMDLINE_LINUX=”” 不改也行
