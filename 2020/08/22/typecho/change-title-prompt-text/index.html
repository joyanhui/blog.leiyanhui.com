在wordpress的自定义模型更改标题输入框提示文字

发表于 2020-08-22

在functions.inc.php或者functions.php增加

/**
 * 更改标题输入框提示文字
 */
function change_default_title( $title )(花括号)
    $screen = get_current_screen();

    if( 'blood' == $screen->post_type ) (花括号)
        $title = '输入证书编号：日月年230502父02母002号01性2';
    (花括号)

    return $title;
(花括号)
add_filter( 'enter_title_here', 'change_default_title' );

注：你可以 post 为其他文章类型，比如，页面为 page。如果你还有其他自定义文章类型，比如 book，也是可以的
灵活地使用 IF 语句进行判断，可以给不同的文章类型设定不同的提示文字，比如：

    if( 'post' == $screen->post_type ) (花括号)
        $title = '输入文章标题';
    (花括号) elseif ('page' == $screen->post_type) (花括号)
        $title = '输入页面标题';
    (花括号) elseif ('book' == $screen->post_type) (花括号)
        $title = '输入书籍标题';
    (花括号)
