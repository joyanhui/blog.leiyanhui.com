百元win10平板改linux系统折腾记录

Posted on 2020-08-22

2020年已更换为Lubuntu20.04
手上几个momo7w平板
几十块到100多块钱咸鱼慢慢收的，本是挂游戏的，现在游戏不玩了废物利用
经过mommo7w技术交流群的大神指点，以及无数次尝试
最终搞定

最终根据自已需求 最终选择 debian10.2 32位 LXQT 环境

之所以选择debian是因为
Ubuntu18.4以后不再支持32位，而且Ubuntu的18.4的32位不支持efi启动。
这板子只支持efi32启动。
加之驱动问题，所以本来是优选Ubuntu
无奈64位的server版虽然搞定了启动，但驱动不会自动安装，本小白没能力和心情去深入研究研究，
desktop版本的不管是16还是18版 虽然都搞定了联网问题，但是安装过程各种卡
死。疑似发热问题导致，不确定

其实心里更喜欢 centos 或者
debian之类的系统，毕竟是玩server。最终搞定debian

选择安装桌面系统是因为不安装的话
它也不给安装驱动，所以就随便选了一个桌面。但是没想到LXQT竟然如此漂亮，如此节省内存。单纯LXQT内存占用应该100M以内，开机不启动LXQT大概50M内存，启动LXQT不登陆
开启一个远程ssh 内存110M左右，登录后打开一个命令行窗口
190M左右。简直和没有一样，所以
也就干脆一直用了，也懒得卸载了。毕竟1G内存的板子明显缺的不是这几十M了

记录过程

测试 兼容momo7w 和 iwork7

准备带充电otg usb hub 键盘 鼠标 4g以上u盘1个
额外准备一个路由器 或者安卓手机，开wifi wifi要求英文名字，免密码
不加密。

1、下载三个东西
1.1下载debian镜像
http://mirrors.163.com/debian-cd/10.2.0/i386/iso-dvd/
只下载dvd 1即可
http://mirrors.163.com/debian-cd/10.2.0/i386/iso-dvd/debian-10.2.0-i386-DVD-1.iso
1.2、下载rufus
https://rufus.ie/
直接下载
https://github.com/pbatard/rufus/releases/download/v3.8/rufus-3.8p.exe
1.3、网卡驱动
https://packages.debian.org/uk/sid/all/firmware-realtek/download
直接提供了 firmware-realtek_20190717-2_all.deb

2、用rufus烧录启动u盘
打开rufus 插入u盘 能自动检测到，选择 debian-10.2.0-i386-DVD-1.iso 镜像
点开始，根据提示自动下载一些兼容性支持，iso模式烧录。
烧录完成后，打开u盘 把1.3的firmware-realtek_20190717-2_all.deb
复制到u盘的firemware文件夹里面，默认里面有一个0字节的deb不用 放一起就好

3、u盘 键盘 接入 momo7w 开机 点esc 选择u盘启动
3.1 选择debian install 那个安装 不用选Graphical debian installer图形界面
图形界面略慢
3.2 搜索WiFi的时候，加密的连不上，需要用无密码的 密码不用输入
获取ip会失败，手动配置 改ip为 192.168.1.200 子网255.255.255.0 网关和dns
192.168.1.1 不同路由器ip段设置不太一样，自己根据情况设置。
安装过程中的wpa2加密和dhcp不可用，具体原因不清楚。所以提供一个没密码的wifi
手动设置ip就可以。
3.3
不安装图形界面的话，网卡驱动也会装不上，具体原因不明。可以选择轻量的lxqt安装
挺漂亮也省内存

批量安装方法 或者直接用我提供的镜像
1、准备两个u盘 4G以上的 鼠标键盘 otg hub
2、U盘A 烧录
http://mirrors.163.com/debian-cd/10.2.0-live/i386/iso-hybrid/debian-live-10.2.0-i386-lxqt.iso
插入平板，启动 默认 等待 进入debianlxqt界面 打开命令 ctrl+alt+t
ctrl+alt+e 是文件管理器
2.1、如果要备份自己的 插入U盘B 然后 命令类似 sudo dd if=/dev/mmcblk1
bs=204800k |gzip > /media/user/12DAC79DDA-usbB/mmc.img.gz 漫长等待
完成
2.2、如果要恢复 把mmc.img.gz放到U盘B以后
sudo -i
sudo gzip -dc /media/user/12DAC79DDA-usbB/mmc.img.gz | dd
of=/dev/mmcblk1 bs=204800k 漫长等待完成
12DAC79DDA-usbB 是你u盘B挂在到debian live后的路径。不同u盘 会不一样。ls
/media/user/ 查看一下就ok
bs=204800k 一定要加上 100M 也行 不然 慢的到怀疑人生
dd 不能直接查看进度，可以另外开一个窗口输入sudo watch -n 5 pkill -USR1
-x dd 这样运行dd的就5秒刷一次进度了

不愿意折腾 disk.img.gz 可以直接使用我提供的momo7w的 也完全兼容iwork7
稍后有时间分享出来。
默认用户名 debian 密码1 内置root 密码也是1
这个步骤1个u盘偶尔会出现不自动加载，准备两个u盘方便点

下面是我做的mmc.img.gz的一些说明和注意点

momo7w debian

基于debian10 LXQT 32位 制作
momo7w 1g＋16g
保留完整功能，内置wifi 常用usb设备 和emmc驱动
wifi驱动为官方版本，emmc驱动为debian10自带 极度稳定
适度精简，有图形界面，有中文系统，有ssh
，包含完整新立德，删除了办公套件，浏览器，输入法，多媒体套件等应用软件，可以自己安装更喜欢的
不带触摸屏驱动(没必要) 蓝牙(没测试)和电池驱动(不影响充电)

内置两个用户
普通用户名debian 密码 1 已经允许sudo
超级管理员 root 密码1

包含lxqt开机，登陆后只占用190m左右内存，未登录的时候只占用117m
server用户完全可以不删除LXQT，或者init 3 禁止启动桌面系统，或者init 5
启动桌面系统。当然也可以更换其他桌面系统，或者直接删除LXQT

默认wifi链接，fish0 密码空，未开启dhcp
ip 192.168.1.200 255.255.255.0 192.168.1.1

更改apt-get源为国内阿里云，更新和安装速度飞快。 /etc/apt/sources.list
全新444，如果需要刷其它源，自己chmod 改下权限

可以自行安装浏览器。桌面readme.txt有提示。
搜狗拼音 qq 火狐 chrome kplayer等所有linux软件都可以自行安装

提示加载内存盘的地方时间比较久，耐心等待大概30到40秒左右才可以。稍后出现一些字符
一直到出现登录界面，用户名输入debian，密码1 登录。开始爽。

===说明===
提示 热键注册 和不同键盘有关系，不影响使用，也可以自己改下热键
没有声音是因为没pulseaudio，如果听歌观影请自行安装
qt的外观主题千万不要选bb10开头的那几个，会导致在自带屏上显示不全，如果万一选了，找个高分的外接显示器改回去,很麻烦
外接屏幕导致自带屏幕不显示的时候，可以在加载内存盘后显示一大串命令行的时候马上拔掉外接屏幕，然后进入桌面后再按电源键重启一次
电池管理驱动让我卸载了，因为准备改4v直接供电，屏幕都打算拆了，所以并不打算用电池。
因为LXQT的优秀 debian的稳定 让我放弃了继续精简的动力，所以此版本
就是最终版本了。
