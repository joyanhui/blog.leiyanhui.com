Win10LTSB长期支持版本安装应用商店

发表于 2020-08-22

LTSB版本默认没装edge也就忍了。但是没有应用商店，我还要玩一个小游戏。。。换系统肯定麻烦了。凑合一下，完美搞定。记录一下操作

1、下载一个win10的镜像，只要64位的就可以了中文的就可以了，我下载了一个多版本。要用install.wim。

2、C:\Program Files\WindowsApps 
这个隐藏的文件夹，修改一下所有人权限为当前管理员用户。确定读写完全控制什么的

3、用7zip或者其他工具打开install.wim把里面任意个分卷的\Program
Files\WindowsApps中的所有内容解压到C:\Program Files\WindowsApps

解压完成后 把WindowsApps的所有者再改为 NT Service\TrustedInstaller
直接输入，搜不到这个用户的。然后修改后
不要点击最后一个确定，返回权限设置对话框后，先不要点确定（否则回到重新给权限），而是点击”添加”-选择主体，查找位置仍为本机，输入”All
Application Packages”。
确定然后保留默认的权限勾选。返回权限设置对话框，勾上”使用可从次对象的权限项目替换子对象的权限项目”，点确定。

4、管理员权限打开Windows Powershell，运行

::: (花括号)#code_Wq0(花括号)
Add-AppXPackage -Register 'C:\Program
Files\WindowsApps\*\AppXManifest.xml' -DisableDevelopmentMode
:::

运行后出现一堆错误，因为依赖包的问题，不用管。中间如果停止了
就点一下回车。然后运行一遍后 看看开始菜单有没有添加应用商店。
没有的话就再运行几次，第一次好像什么的也没装上，第二次装了几个应用但是没有商店，第三次就出来商店了，其实还可以再运行几次，一直到不出错所有的应用都安装。多运行几次
出来多装一个默认应用没什么坏处。

5、卸载掉没什么的用的应用 比如天气 纸牌之类的。

试试应用商店。

记得使用应用商店 要打开Windows update服务

把ipad作为电脑的扩展显示器使用（安卓平板iPhone也可以）-拒绝收费app拒绝广告app

发表于 2020-08-22

上一篇文章说了
ipad和iPhone镜像到win10的办法
    这次说 怎么镜像或者扩展ipad做电脑的显示屏。

不要付费买那些垃圾软件，还要拖一个usb线 还延迟。<!--more-->

AppStore 搜一圈都几十块
几百块，还要托一根线，只支持mac，不支持win10。。。。。

MirrorOp MirrorSender Duet Display 等一堆app
好不容找一个能用的发现只能复制屏幕，那还不如直接用teamviwer的局域网远程

先把pc和ipad接入同一个路由器。这个不用多说。

然后说扩展功能。我本来打算也用teamviewer的后来发现win10下找不到虚拟的显示屏，所以，就放弃了。

然后找到了spacedesk这个软件，支持从win端扩展或者复制到全平台。win7-win10
可以投屏到 安卓 ios win等系统。最牛逼的是 支持html5
，也就是客户端只要有一个浏览器 就可以。 
而且可以无限制扩展N个屏幕，有几个客户端 就有几个屏幕。

pc端下载：https://www.spacedesk.net/

ipad应用商店直接搜：spacedesk

都安装完成打开后，pc端 win+P 扩展显示 或者复制  完事了

超级简单，一分钱不花速度嗷嗷的！！！

ipad一根线都不用，拿到桌子上打开app就可以用,轻松续航8-10小时

最后一个好玩的分享，打开浏览器新建N个标签 
然后打开speacdesk的html5客户端，你懂的。。。n个屏幕扩展。。。

因为ipad的分辨率比较高，还是要设置一下屏幕缩放，不然拓展模式显示的字体确实很小

2018-10-23更新发现一个bug，有时候频繁断开连接的情况下，ipad上显示的区域非全屏，不过重启一下电脑就ok了

延迟确实存在，毕竟wifi通讯。不过，适应性一下啊 还是ok的
ios端也可以设置流畅度

看视频的话最好降低一下画面质量。码字的时候
有时候也会有感觉延迟，也是降低一下质量解决。另外5Gwifi下体验会提升很多
