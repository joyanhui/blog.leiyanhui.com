百元win10平板改linux系统折腾记录

发表于 2020-08-22

2020年已更换为Lubuntu20.04
手上几个momo7w平板
几十块到100多块钱咸鱼慢慢收的，本是挂游戏的，现在游戏不玩了废物利用
经过mommo7w技术交流群的大神指点，以及无数次尝试
最终搞定

最终根据自已需求 最终选择 debian10.2 32位 LXQT 环境

之所以选择debian是因为
Ubuntu18.4以后不再支持32位，而且Ubuntu的18.4的32位不支持efi启动。
这板子只支持efi32启动。
加之驱动问题，所以本来是优选Ubuntu
无奈64位的server版虽然搞定了启动，但驱动不会自动安装，本小白没能力和心情去深入研究研究，
desktop版本的不管是16还是18版 虽然都搞定了联网问题，但是安装过程各种卡
死。疑似发热问题导致，不确定

其实心里更喜欢 centos 或者
debian之类的系统，毕竟是玩server。最终搞定debian

选择安装桌面系统是因为不安装的话
它也不给安装驱动，所以就随便选了一个桌面。但是没想到LXQT竟然如此漂亮，如此节省内存。单纯LXQT内存占用应该100M以内，开机不启动LXQT大概50M内存，启动LXQT不登陆
开启一个远程ssh 内存110M左右，登录后打开一个命令行窗口
190M左右。简直和没有一样，所以
也就干脆一直用了，也懒得卸载了。毕竟1G内存的板子明显缺的不是这几十M了

记录过程

测试 兼容momo7w 和 iwork7

准备带充电otg usb hub 键盘 鼠标 4g以上u盘1个
额外准备一个路由器 或者安卓手机，开wifi wifi要求英文名字，免密码
不加密。

在一个超小的vps上搭建nginx+php + ssr+frp内网穿透的并共存。

发表于 2020-08-22

128M的vps是某宝购买(已经跑路)，日本ip速度还可以，价格5元/月。年付3.5元左右。比买一个未批嗯
或者花生壳之类的 还便宜。搭建了几个常用服务玩玩<!--more-->
基本配置
1 CPU，RAM 128MB，vSWAP 32MB，硬盘 1.5GB SSD，带宽10Gbps，月流量50GB。
没有独立ip，可选ipv6。
装了debian-wheezy-i386.
80端口是肯定没有不过卖家可以给做转发，也没说能转发几个。除了ssh端口意外，还给了20个公开的端口，足够日常使用了。

这个配置安装mysql有一些扯淡了，而且我一直也不怎么喜欢mysql，所以干脆不装。sqlite还是要的。

ssr是未批嗯的一个替代品，好像也叫影梭，没有仔细研究，大概是加密和混淆，然后让运营商和中间环节不知道你传输的什么。混淆的功能，如果是在80端口上，可以实现移动上网免流量。和以前用的ss代理有点接近的味道。

frp是一个内网穿透软件。大致原理就是
服务器端监听一个端口，客户端和服务端握手后，服务端会把公网的请求分发给客户端。然后对外提供web
ssh或者其他tcp utip之类的服务。

之前也都做了一个记录，现在汇总一下

ssr 需要端口公开的端口 这里使用1901
frp 也需要一个公开端口 使用1902
frp
还需要一个web端口，为了方便调试也分配了一个公开端口毕竟有20个呢，毕竟富裕，这里用1908
nginx 自然是监听80端口，然后反响代理frp的web端口1908就好了。

详细

apt-get install php5-fpm
apt-get install php5-sqlite

                try_files $uri =404;

                fastcgi_index index.php;
                fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
                include fastcgi_params;

        (花括号)
echo '<?php phpinfo(); ?>'>>/usr/share/nginx/www/info.php

service php5-fpm restart
#测试
curl http://localhost/info.php

MacBook Pro 2011 升级安装 macOS catalina 10.15.3 记录

发表于 2020-08-22

MacBook Pro (13-inch, Early 2011) 古董 办公 音视频加轻量编程
足够。但是最高升级10.13，羡慕catalina的暗夜模式。于是尝试升级。
网上搜索的教程要么千篇一律，要么无法复原操作继续进行下载，所以记录下备用

自然是选择 大名鼎鼎的 macos catalina patcher。
1、准备工作 ，当然是先备份，用时间机器
先备份系统到移动硬盘。刚好有一个1T的机械硬盘 备份了一下。
升级前macos版本是10.13.3。
然后升级catalina一直出错，本想放弃。然后就升级了10.13.6暂用。后来结果成功了。
2、下载macos catalina patcher
官网 http://dosdude1.com/catalina/
更新记录 http://dosdude1.com/catalina/changelog.html
我使用的版本1.3.0 - 12/11/2019
下载地址被墙了，找一个梯子下载吧。国内其他地址不知道是版本号问题
还是文件损坏导致莫名其妙的失败，浪费时间。
3、下载后运行
首次运行 会提示无法运行，这时候打开  >系统偏好设置 安全和隐私 允许
macos catalina patcher 允许
注意：必须打开过一次，安全和隐私里面 才会有 macos catalina
patcher，不然看不到的哦。
4、选择下载，8.9g左右，等待下载
5、下载完成后
有提示说明之类的按照提示操作，拔掉移动硬盘，直接选本机安装。然后同意协议
等待重启，重启后需要比较长的时间，尤其是最后提示
1-2分钟的时候，实际需要10分钟左右。
6、不出意外的话，顺利安装完毕。完成基本设置后，在弹出的 patcher
updater里面升级一些新的驱动。再次重启。

=========
还要进恢复模式 关闭一个安全设置，具体看macos catalina
patcher的提示，稍后有时间折腾我再看一下具体是什么设置。

不建议从别的地方下载系统镜像，以免有莫名其妙的问题，另外u盘或者移动硬盘方式安装虽然也支持，但是也出现过无法重启的青睐。也不建议。

使用感受：
没有别人说的发热 和电池衰减的问题。也没有感觉明显的卡顿，
wps 微信 输入噶 Safari vsc qq之类的都正常
很流畅。photoshop要重新装哈，破解的授权失效了。
可能是因为屏幕的原因，字体有一些发虚，回头再研究一下怎么调整 补到本文

不过使用时间还不长，无法准确判断。

字体发虚的问题
字体发虚好像是catalina真对Retina的屏幕的优化效果，但是在mbp2011上明显不可能
办法是启用四种字体的子像素抗锯齿模式

defaults -currentHost write -globalDomain AppleFontSmoothing -int 1
defaults -currentHost write -globalDomain AppleFontSmoothing -int 2
defaults -currentHost write -globalDomain AppleFontSmoothing -int 3
defaults -currentHost write -globalDomain AppleFontSmoothing -int 0

注销后有效果，但是部分字体还会有点，继续研究中》。。

强制打开 sidecar
运行后，在系统编号设置 里面 会有 随行
defaults write com.apple.sidecar.display AllowAllDevices -bool true;
defaults write com.apple.sidecar.display hasShownPref -bool true; open
/System/Library/PreferencePanes/Sidecar.prefPane

然后重启。

本来是看了一个游戏里面的朋友做的乐透脚本，然后发现原来写一个小辅助那么简单。本来就会一些编程，所以一边玩游戏一边看手册写代码完善。
最后完成了这个游戏里面的所有的操作类的函数，而且趋近完美。

详细说明：
一、自动采矿功能
a)在指定矿区内采集，不会越界
b)可单独采集某一种矿，或者多种矿。
i.支持 粮、木、石、铁、银五种不支持金矿
ii.识别率在80%以上
iii.最快15秒搜索完成一个矿区
iv.可跳过矿区内正在被偷的矿
c)可使用预设队列II和预设队列II，实现指定的兵种采指定的矿。例如剑兵挖木、骑兵挖石、铁
d)可以优先采集焦点矿
i.需要提前录入焦点矿区坐标
ii.焦点矿区内会尝试采集设置了焦点矿的矿类型
e)可以分散采矿，防止集中下矿
f)长时间采集不到可以自动重启
g)不支持矿等级选择，优先采集先找到的矿，不优先采集高级矿
h)自动检查采集加速25%的药，剩余时间1小时左右会自动补药
i)自动纠错功能
i.界面错误会尝试重置
ii.识别错误会自动跳过
iii.长时间采集不到矿会重启游戏
iv.掉线会自动重新连
v.游戏卡死自动重启游戏
二、被打矿自动撤矿功能
a)可以检测敌军行军队列
b)可以检测战报
c)被打矿后可以设置随机的时间重新开始
三、自动清理间谍功能
a)可以在最后5分钟内清楚所有间谍，避免间谍骚扰 四、自动点盟友帮助
五、自动续盾功能
a)提前30-50分钟自动续盾
b)可以设置续8小时盾或24小时盾
c)掉盾危险，此功能仅为辅助，切勿依赖
六、快速刷乐透功能
a)支持普通乐透和活动乐透，不支持皇家乐透
b)支持开箱子功能
i.支持箱子只开一个
ii.支持打开所有箱子
iii.支持继续打开箱子，直到开打3天加速
c)乐透刷完可以继续其他操作，例如：刷完乐透自动去采矿
七、扫死城半自动辅助功能
a)此功能兼容所有模式和分辨率
b)人工点击死城后，程序帮助点：攻击>>选择预设兵>>留下斗士>>出发攻击
更快
八、自动关机功能
a)使用外部小软件实现
b)支持倒计时或者定时，或者每天：关机、重启、注销
九、手机、平板远程控制功能
a)和向日葵软件完美配合
b)远程暂定或者关闭后，不影响手机登陆
十、兼容性
a)只支持1920*1080以上分辨率 ，新版已经支持所有分辨率
b)兼容99%的电脑显卡
c)兼容win10
十一、可以根据电脑配置设置运行速度
a)高配置电脑请选择最快，否则选择慢
b)刷乐透和辅助扫成可快到看不清鼠标
十二、暂结待续2018-1-25

发表于 2020-08-22

当然是首选图形界面的，想过宝塔 但是要用docker
有点折腾了。轻量开发而已。。懒得搞。于是决定用自带的apache和php，也放弃了nginx。
省的折腾，反正就是一个轻量开发环境。apache和nginx对我来说无差

最后我还是换了虚拟机装了宝塔，Docker宝塔很不方便，占用硬盘也不小。暂时用的pd，回头有时间换Vmware

1、准备和安装
1.1、查看apache和php版本

我的是 Apache/2.4.33 (Unix) PHP/7.1.16 macos10.13.6

1.2、安装mysql
，有好多版本没有macos版，本来打算安装5.5的懒得找，随便找了一个5.7
https://downloads.mysql.com/archives/get/p/23/file/mysql-5.7.9-osx10.10-x86_64.dmg
安装完成会提示 默认密码
2020-03-10T06:30:11.847539Z 1 [Note] A temporary password is generated
for root@localhost: )W_w1:=j>trV
安装后系统》偏好设置，启动
cd /usr/local/mysql/bin/
./mysql -uroot -p
修改密码
set password for root@localhost = password('123');
其他版本从 https://downloads.mysql.com/archives/community/ 下载
2、修改apache支持php
sudo code /etc/apache2/httpd.conf
如果找不到code可以用vi、edit或者在vscode中shift + command + p
输入shell选择安装code到shell
找到 #LoadModule php7_module libexec/apache2/libphp7.so
去掉注释
3、修改apache默认路径
找到
DocumentRoot "/Library/WebServer/Documents"
<Directory "/Library/WebServer/Documents">
修改为 /Users/yanhui/www/htdocs 保存
sudo mkdir -p /Users/yanhui/www/htdocs
sudo chown -R _www:_www /Users/yanhui/www/htdocs
sudo chmod -R 777 /Users/yanhui/www/htdocs
创建一个文件
echo "<?php phpinfo(); ?>" >> /Users/yanhui/www/htdocs/index.php
下载phpmyadmin
4、启动和测试
开启Apache：sudo apachectl start
关闭Apache：sudo apachectl stop
重启Apache：sudo apachectl restart
