详细说明：
一、自动采矿功能
a)在指定矿区内采集，不会越界
b)可单独采集某一种矿，或者多种矿。
i.支持 粮、木、石、铁、银五种不支持金矿
ii.识别率在80%以上
iii.最快15秒搜索完成一个矿区
iv.可跳过矿区内正在被偷的矿
c)可使用预设队列II和预设队列II，实现指定的兵种采指定的矿。例如剑兵挖木、骑兵挖石、铁
d)可以优先采集焦点矿
i.需要提前录入焦点矿区坐标
ii.焦点矿区内会尝试采集设置了焦点矿的矿类型
e)可以分散采矿，防止集中下矿
f)长时间采集不到可以自动重启
g)不支持矿等级选择，优先采集先找到的矿，不优先采集高级矿
h)自动检查采集加速25%的药，剩余时间1小时左右会自动补药
i)自动纠错功能
i.界面错误会尝试重置
ii.识别错误会自动跳过
iii.长时间采集不到矿会重启游戏
iv.掉线会自动重新连
v.游戏卡死自动重启游戏
二、被打矿自动撤矿功能
a)可以检测敌军行军队列
b)可以检测战报
c)被打矿后可以设置随机的时间重新开始
三、自动清理间谍功能
a)可以在最后5分钟内清楚所有间谍，避免间谍骚扰 四、自动点盟友帮助
五、自动续盾功能
a)提前30-50分钟自动续盾
b)可以设置续8小时盾或24小时盾
c)掉盾危险，此功能仅为辅助，切勿依赖
六、快速刷乐透功能
a)支持普通乐透和活动乐透，不支持皇家乐透
b)支持开箱子功能
i.支持箱子只开一个
ii.支持打开所有箱子
iii.支持继续打开箱子，直到开打3天加速
c)乐透刷完可以继续其他操作，例如：刷完乐透自动去采矿
七、扫死城半自动辅助功能
a)此功能兼容所有模式和分辨率
b)人工点击死城后，程序帮助点：攻击>>选择预设兵>>留下斗士>>出发攻击
更快
八、自动关机功能
a)使用外部小软件实现
b)支持倒计时或者定时，或者每天：关机、重启、注销
九、手机、平板远程控制功能
a)和向日葵软件完美配合
b)远程暂定或者关闭后，不影响手机登陆
十、兼容性
a)只支持1920*1080以上分辨率 ，新版已经支持所有分辨率
b)兼容99%的电脑显卡
c)兼容win10
十一、可以根据电脑配置设置运行速度
a)高配置电脑请选择最快，否则选择慢
b)刷乐透和辅助扫成可快到看不清鼠标
十二、暂结待续2018-1-25

1、准备和安装
1.1、查看apache和php版本

2、修改apache支持php
sudo code /etc/apache2/httpd.conf

找到 #LoadModule php7_module libexec/apache2/libphp7.so

找到
DocumentRoot "/Library/WebServer/Documents"
<Directory "/Library/WebServer/Documents">

sudo mkdir -p /Users/yanhui/www/htdocs
sudo chown -R _www:_www /Users/yanhui/www/htdocs

关闭Apache：sudo apachectl stop

linux下自动同步备份ftp空间

发表于 2020-08-22

可以在路由器或者vps中运行，实现自动备份网站ftp文件。
直接使用wget<!--more-->

mkdir /home/wwwroot/from_aliyun_web
cd /home/wwwroot/from_aliyun_web
#删除10天之前的备份
find /home/wwwroot/from_aliyun_web -mtime +10 -name "bak-from*" -exec rm -rf (花括号)(花括号) ;
rm -rm ./*.sql
#下载
wget -nH -m --ftp-user=  --ftp-password= ftp://.ftp.aliapp.com/htdocs/ 
#打包
tar -zcvf bak-from-aliyun-web-$(date +%Y%m%d-%H%M).tar.gz htdocs/
#mysqldump 没装的话，需要yum install mysql
mysqldump -P 3306 -h rm.mysql.rds.aliyuncs.com -u[root] -p[passwd] [table]>/home/wwwroot/from_aliyun_web/mysql-.sql
#打包
tar -zcvf bak-aliyun-mysql-.sql-$(date +%Y%m%d-%H%M).tar.gz mysql-.sql

chmod +x /root/auto/ftp_get.sh
crontab -e
01 02 * /root/auto/ftp_get.sh
